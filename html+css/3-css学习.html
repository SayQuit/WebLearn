雷门中学的足球部是一个仅有7名部员的弱小队伍。社团活动室内聚集了一批拥有各自爱好,度过糜烂每一日的部员。 热血足球少年的队长·円堂守激励部员们“为了明天的比赛练习吧!”。其中居然有强队帝国学园那里发来的比赛邀请。 但是要比赛的话还缺少4人。而且如果不参赛的话将会关系到足球部的存活。所以无论如何为了要实现与帝国学园的比赛, 円堂开始致力于一个人练习。某一天,円堂与拥有必杀射门技“火焰龙卷”的“传说中的前锋”豪炎寺修也相遇了,且游说他入队。 其它成员找齐了。在比赛当日円堂守他们以大比数落后,在一位队员退场以后,豪炎寺修也代替他加入比赛并进了一球。 最后帝国学园竟然放弃比赛,因此算是雷门中获胜。为此雷门中学足球部得以保留,这班足球少年又可以向全国大会进发。
